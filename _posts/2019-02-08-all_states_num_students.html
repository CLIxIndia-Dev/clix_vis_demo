In a given school, number of students engaged with different tools in a day is calculated. This number for each tool is averaged over the days on which that particular tool is used. Chart should be used to compare number of students engaged with different tools in a school, but not to deduce total students using tools in a day (cannot add up number of students along y-axis). calculations
