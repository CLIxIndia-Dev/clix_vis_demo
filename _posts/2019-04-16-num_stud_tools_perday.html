These visuals indicate number of students engaged with different tools in a typical day. In a given school, number of students engaged with different tools in a day is calculated. This number for each tool is averaged over observation period. Chart should be used to compare number of students engaged with different tools in a typical day, but not to deduce total number of students using tools in a day by adding up numbers on y-axis. Only logged-in users are considered, anonymous users are counted as only one student. calculations
